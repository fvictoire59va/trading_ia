# Trading IA

Projet de trading algorithmique avec intelligence artificielle.

## Description

Ce projet implémente un système de trading automatisé utilisant des techniques d'intelligence artificielle pour analyser les marchés financiers et prendre des décisions de trading.

## Prérequis

- Python 3.9+
- Docker
- Docker Compose

## Installation

```bash
# Cloner le dépôt
git clone https://github.com/fvictoire59va/trading_ia.git
cd trading_ia

# Installer les dépendances
pip install -r requirements.txt
```

## Utilisation

```bash
# Démarrer avec Docker
docker-compose up -d
```

## Structure du projet

```
.
├── src/                # Code source
├── data/              # Données de marché
├── models/            # Modèles d'IA entraînés
├── tests/             # Tests unitaires
└── docker/            # Configuration Docker
```
